PERGUNTAS FREQUENTES:

"Preciso pagar para utilizar o site?"

Não. Você só realizará o pagamento do agendamento.

"É necessário criar uma conta para utilizar o site?"

Não. Você só precisará de uma conta, caso deseje agendar um horário.

"Qual o valor de agendamento?"

O valor de agendamento é definido pela empresa, portanto, pode variar.

"Posso cancelar um agendamento?"

Sim. Porém você deve respeitar as condições de cancelamento da empresa.

"Quais métodos de pagamento são aceitos?"

Cartão de crédito, débito e pix via plataforma. Dinheiro presencialmente.

"Como faço para implementar meu serviço na plataforma?"

Entre em contato consco para debatermos nossas ofertas.
